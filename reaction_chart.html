Spell Reaction Chart. Botter's status-combo system, DOS2-style: one element hitting an enemy that already carries a status triggers a reaction. Because the bot plays itself, you author combos at the loadout screen — equip spells whose statuses feed each other — and read the payoff in the combat log/VFX. This is the ratified core (see docs/spell-mechanical-diversity-plan.md); it's the design spec for Phase-3 combat, not all shipped yet. Read the grid: row = the element you hit with, column = the status already on the target, cell = the reaction. Click any reaction for detail.
