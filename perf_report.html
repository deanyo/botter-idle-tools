What shipped

Every fix is a real code change with a file:line in the checklist. Tier 0 = playtest blockers or S-effort quick wins. Tier 1 = P1 with M-effort. See docs/perf-audit-2026-07-01/PERF_CHECKLIST.md for full tier breakdown.

Tier 0Glow-material cache keyed on color only
bot.gd:1088 · paperdoll_renderer.gd:154
Slider drags on FX Tuner used to compile a fresh Web GL program per step because strength and thickness lived in the cache key. Now uniform mutations on the cached material — no recompile.
Tier 0VideoSettings.load_settings cache
video_settings.gd:203
Combat-hot config reads (tunable(), is_effect_enabled()) fired 15-25× per frame and re-parsed user://video_settings.json from disk each time. Now O(1) Dict lookups; cache invalidated only on save_settings().
Tier 0Fog nearest-8 light sort
fog_overlay.gd:126
Web-GL fragment cost caps active lights at 8. Pre-fix picked arbitrary 8; now sorts world lights by distance-to-bot before the slice — near-bot torch always wins.
Tier 0Autosave chain redundant reload
dungeon.gd:2603
Autosave was doing flush_to_save() then SaveState.save_state(SaveState.load_state()) — 3 JSON parses + 2 writes per 60s tick. Dropped the redundant reload.
Tier 0Boss + portal texture prewarm
biome_data.gd:151 · dungeon.gd:2436 · outpost.gd:187
Boss textures lived outside enemy_pool and portal biomes were never in the outpost's unlocked walk. Now prewarmed proactively at Outpost across all Portal.PORTAL_KINDS and defensively at portal entry.
Tier 0Initial spawn stagger
dungeon.gd:5251 · wave_spawner.gd
Pre-fix _spawn_packs synchronously instantiated up to 350 mobs on floor build. Now spawns pack leaders synchronously (floor reads populated immediately) and queues packmates into wave.pending_initial_spawns drained 4/frame from _process.
Tier 1Autosave retry-delay skip
save_state.gd:1024 · run_state.gd:420
_save_wrapper gained retry_delay_ms. Autosave path passes 0 — no OS.delay_msec(1000) stall in _process on transient FS contention. Quit/menu/shop keep the 1s default.
Tier 1Compact save JSON
save_state.gd:1082
Dropped 2-space indent from JSON.stringify on save. Verified live save file shrunk ~14KB → ~5.7KB (60% reduction). Halves stringify+parse cost.
Tier 1Migration walker across all containers
save_state.gd::_walk_all_item_containers
_migrate_to_v8/v10/v13 now route through one helper that hits equipped + inventory + stash + orphaned_items. Pre-fix, several migrations skipped stash and orphaned_items — Steam-Cloud-tanking silent stat loss.
PlaybookRules 1, 3, 4 amended + new Rule 7
CLAUDE.md — HTML5 perf playbook
Rule 1 (cache-key vs uniform mutation), Rule 3 (data-driven prewarm — walk JSON, no hand-lists), Rule 4 (ms-budget stagger), new Rule 7 (config reads are hot-path). Post-amend re-sweep found three new drift sites (portal gateways, ambient decor, spell projectile textures) — all fixed inline.
